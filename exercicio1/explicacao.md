# Exercício 1 – Preparação do ambiente

## O que foi feito
Instalação e verificação do OpenSSL no ambiente WSL2 (Ubuntu).

## O que é o OpenSSL?
OpenSSL é uma biblioteca de código aberto que implementa os protocolos
SSL (Secure Sockets Layer) e TLS (Transport Layer Security), além de
fornecer ferramentas de criptografia de uso geral.

Com ele é possível:
- Gerar chaves criptográficas (RSA, ECDSA, etc)
- Criar e assinar certificados digitais (X.509)
- Cifrar e decifrar dados
- Verificar a integridade de arquivos com hashes
- Simular conexões SSL/TLS para diagnóstico

É a ferramenta padrão utilizada por servidores web (Apache, Nginx) e
sistemas operacionais para gerenciar certificados e conexões seguras.
